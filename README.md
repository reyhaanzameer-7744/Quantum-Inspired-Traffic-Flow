🚦Quantum-Inspired-Traffic-Flow
The Problem
Traditional traffic lights use fixed timers, causing unnecessary waiting, fuel waste, and high CO2 emissions.

The Solution
This project uses Quantum-inspired optimization to analyze traffic queues in real-time and prioritize the busiest lanes, including an Emergency Ambulance Override.

Tech Stack
Python (Core Logic)

Streamlit (Web Interface)

Plotly (Live Animations)
